# DLS
Assignments covered during Deep Learning for Search course

### — Basics of Natural Language Processing 

Text preprocessing, tokenization, the Bag of Words model, cosine similarity

### — K-Means Clustering 

K-Means Clustering using NumPy. Embeddings generation, clusterization on a dataset with different ranking metrics

### — Ranking Metrics 

Implement ranking metrics using NumPy (no loop iteratons), model search outputs, investigate metrics’ behavior.

### —Deep Learning and Embedding 

Similarity search, dimensionality reduction tasks on embeddings, search queries.
